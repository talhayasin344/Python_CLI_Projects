## ▶️ Running a Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/python-cli-projects.git
```

2. Navigate to the project folder:

```bash
cd python-cli-projects
```

3. Open any project directory:

```bash
cd project-name
```

4. Run the project:

```bash
python main.py
```

or

```bash
python3 main.py
```
