## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/talhayasin344/Python_CLI_Projects.git
```

### Navigate into the repository

```bash
cd Python_CLI_Projects
```

### Open any project folder

```bash
cd project-name
```

### Run the project

```bash
python main.py
```

or

```bash
python3 main.py
```
